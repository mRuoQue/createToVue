# createToVue __ 创建vue3模版脚手架

* 1 创建v3项目目录
* 2 创建项目结构
* 3 下载项目初始化依赖
* 4 进入项目，帮助启动项目
    > 主要目前方便自己使用，不喜欢vite创建的格式，抽空会支持其他框架，并且简化创建流程.
